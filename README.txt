Visualization shows indicators used in global health surveys, separated into female indicators, verb indicators, and male indicators

Parent Class: WordCloud, Children: FemaleCloud and MaleCloud
- found in word-cloud.js
- MaleCloud method drawCloud() overrides drawCloud() of parent class

- RiTa is used in main.js
